
## Vue project that uses:
* Node.js
* Express
* Browser Sync
* Browserify
* Babel (ES 6 [ES 2015])
* Vue
  * Vue Router
  * Pug
  * Stylus
* Unit Testing
  * Karma
  * Jasmine
* JS Linter (EsLint)

## To Run:
* Clone Repo
* Install Node
* Install NPM
* Run the install (`npm install`)
* Run Gulp (`gulp`)

## To Test:
* Run `gulp test` - Also runs when you run `gulp`
