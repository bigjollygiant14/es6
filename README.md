
## Vue project that uses:
* Node.js
* Browserify
* Babel (ES 6 [ES 2015])
* Vue
  * Vue Router
  * Pug
  * Stylus

## To Run:
* Clone Repo
* Install Node
* Install NPM
* Run the install (`npm install`)
* Run Gulp (`gulp`)
